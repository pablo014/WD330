JS Arrays and Loops

JavaScript has different ways of creating arrays. They are assigned differently because JavaScript does not use defined variables but defines it when assigned. So the first way of assigning a an array is using the [] symbol. For example...

var objects = [jukebox, soup, cereal];

This is an array that has been filled out, however if you want to create an empty array you would define it like this.

var array = []

JS Functions

JavaScript has 3 ways of defining a function.

Regular Function Declaration

function func() { var variable = "This is a variable"; }

Anonymous Function Declaration

const func = function() { var variable = "This is a variable"; }

Arrow Notation

const func = () => { var variable = "This is a variable"; }
