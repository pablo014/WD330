There are a few ways to debug in Javascript. The main ways to do so are these

  • console.log()
  • alert()
  • DOM

console.log() allows the user to place a value within it. This will then print it into the console so that you can see it. This example function is a small example of testing and debugging using console.log()

var string = "Inside debug" function test() { debug(); } function debug(){ console.log(string) }

alert() allows the user to see the result when the website is running. It will show a pop up when the script is run

function debug() { alert('inside debug') }

You can also place your debug inside the DOM, however it is not advised. Here is an example

practice html (see in inspect)

practice js

function debug() { document.querySelector('div').innerHTML = "inside debug" }
