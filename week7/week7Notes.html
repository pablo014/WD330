AJAX Requests

AJAX means "Asynchronous Javascript and XML". This type of request works within a Javascri[t file and receives parameters just like any function would and returns a JSON string. In W3Schools it says "AJAX is a developer's dream, because you can:

  • Update a web page without reloading the page
  • Request data from a server - after the page has loaded
  • Receive data from a server - after the page has loaded
  • Send data to a server - in the background

An AJAX request has a couple components to it, the HTML page and the Javascript. The html page has simply a button or method that loads the AJAX request in Javascript. The Javascriptlooks like this

function loadDoc() { var xhttp = new XMLHttpRequest(); xhttp.onreadystatechange = function() { if (this.readyState == 4 && this.status == 200) { document.getElementById("demo").innerHTML = this.responseText; } }; xhttp.open("GET", "ajax_info.txt", true); xhttp.send(); }

Here are the parts of the AJAX request. The declaration of a new AJAX request, the action, opening the ajax request, and sending the request.
