Forms

Forms are a way for a user to fill out and submit information. A user can use a form to fill out and submit. A form can be created by using a form tag.

The form tag usually has two parts of the tag, the action and the method. The action usually would lead to an action page and the method is how it will send the information.

Within forms you can add inputs that allows the user to insert information.

There are many forms of inputs that can be researched on how to make them. Some of them can be found in W3Schools.com
