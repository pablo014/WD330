Week 3 Notes

This

The this keyword is a keyword that specifies an object and it changes within the context. There are differences in the global, object, and function frame

Global

console.log(this)

You probably will never use this, because the information in here isn't in too much importance

Object

function Human (age) { this.age = age } let greg = new Human(22) let thomas = new Human(24) console.log(greg) // this.age = 22 console.log(thomas) // this.age = 24

Function

function simpleFunction () { console.log(this) } const o = { sayThis () { simpleFunction() } } simpleFunction() // Window o.sayThis() // Window

Use this in functions to remain in the same method

Objects

Objects can be created in multiple ways

\ var myCar = new Object(); myCar.make = 'Ford'; myCar.model = 'Mustang'; myCar.year = 1969;

or

var myCar = { make: 'Ford', model: 'Mustang', year: 1969 };

To add a method you can do this

var example = { name: myName //this is called a property sayHello() { //this is a method console.log("Hello"); } }
