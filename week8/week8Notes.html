Transitions and Transformations

Transitions and transformations are a powerful way to, as the name entails, alter the look of the page. The css code for tranistioning anything is this

div { transition: [property] [duration] [timing-function] [delay]; }

Transition Property

The transition property is required and tells it what to do ie. enlarge, shrink, etc.

Transition Duration

Transition duration tells you how long it will take to acccomplish that task and is also required. This parameter can be specified in milliseconds or seconds

div { transition-duration: 3s; }

Transformations

While transitions move the object or change its size, transform changes the element completely. Transform can mainly do things like skew an element or translate it

div { transform: skewX(25deg); transform: skewY(10deg); transform: skew(25deg, 10deg); }

Canvas, SVG, and Drag and Drop

These functions are different dynamic functions done on html using javascript

Canvas

The canvas property allows you to draw on the page dynamically and on the fly. See below

SVG

SVG stands for Scalable Vector Graphics and it defines vector based graphics in XML. You can create and edit shapes using this.

Drag and Drop

Drag and drop is a system that combines html and javascript in order to dynamically move an element by dragging and dropping
